PLACE YOUR MODEL FILE HERE
==========================

Copy your trained model file into this folder:

    skin_cancer_model.tflite

The model must:
- Be in TensorFlow Lite format (.tflite)
- Accept input shape: (1, 128, 128, 3) float32
- Output shape: (1, 7) softmax probabilities
- Classes order: ['akiec', 'bcc', 'bkl', 'df', 'mel', 'nv', 'vasc']

See README.md for full setup instructions.
